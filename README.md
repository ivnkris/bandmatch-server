## Getting Started

### Installation

```
git clone https://github.com/invkris/bandmatch-server.git
cd bandmatch-server
npm i
```

### Data Set

To seed the data:

```
npm run seed
```

### Launch the server

```
npm run start
```

## Technologies Used

For this project we used the following technologies:

- Apollo Server Express
- GraphQL
- MongoDB + Mongoose
